About PrEYEvacy

PrEYEvacy is a search engine that uses other search engines, pretty nifty huh?

More about PrEYEvacy...

    Why use PrEYEvacy?

    • PrEYEvacy may not offer you as personalised results as Google, but it doesn't generate a profile about you.
    • PrEYEvacy doesn't care about what you search for, never shares anything with a third party, and it can't be used to compromise you.

    If you do care about privacy, want to be a conscious user, or otherwise believe in digital freedom, make PrEYEvacy your default search engine or run it on your own server

    How to set as the default search engine?

    PrEYEvacy supports OpenSearch. For more information on changing your default search engine, see your browser's documentation:

    • Firefox
    • Microsoft Edge
    • Chromium-based browsers only add websites that the user navigates to without a path.
